2. Description of service

The Extension is an unofficial, independent browser extension that adds organisational features to the claude.ai web interface. It is not affiliated with, endorsed by, or produced by Anthropic, PBC. The Extension:

  • Reads conversation titles from the claude.ai interface to enable search and organisation
  • Stores user-created organisational metadata (folders, tags, notes) in local browser storage
  • Optionally syncs this metadata to cloud servers for Pro subscribers
  • Provides a prompt library for storing and inserting frequently used prompts
  • Provides an export feature that generates Markdown files from visible conversation content
